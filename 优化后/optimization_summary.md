## 关键技术改进

### 1. 目标值归一化（R9）
将测试阳性率归一化到均值0、标准差1，稳定梯度流，loss下降更平稳。

### 2. 数据划分优化（R11）
从 90/10 改为 80/20 划分：
- 更多验证样本（540 vs 270）→ 更可靠的早停判断
- 验证集更充分代表数据分布

### 3. 网络架构改进（R12-R14）
- **更宽**：256→128→64→32（原 128→64→32）
- **更好激活**：GELU（原 ReLU），平滑非线性，训练更稳定
- **更轻正则**：Dropout 0.10（原 0.30），GELU 自带隐式正则

### 4. 多模型集成（R16）
3 个不同随机种子（42069, 12345, 67890）的模型预测取平均：
- 各模型 Dev MSE: 0.6898 / 0.7030 / 0.7041
- 集成降低预测方差，提升 Kaggle RMSE

## 最终模型架构

```
Input(93) → Linear(256) → GELU → Dropout(0.10)
         → Linear(128) → GELU → Dropout(0.10)
         → Linear(64)  → GELU → Dropout(0.08)
         → Linear(32)  → GELU → Dropout(0.05)
         → Linear(1)
```

- 优化器：AdamW (lr=0.001, weight_decay=1e-3)
- 调度器：CosineAnnealingWarmRestarts (T_0=80, T_mult=2, eta_min=1e-7)
- 早停：400 epochs 无改善
- 批量大小：128
- 数据划分：80/20
